Web page written by Matthew Thomas Kelly.

This is a Python Django powered site, with elements of Bootstrap.

Submission of the search form utilises Python's beautiful soap, URL and JSON libraries.

It was written as a tool to aid Police Officers in searching for items accross multiple auction websites and is easily adaptable to include further auctions sites than those currently supported. It has also been written in such a way to allow the web-scraping aspect to be simply amended in the event of target website structure changes.

The code is available on the GitHub site of Matthew Thomas Kelly.
